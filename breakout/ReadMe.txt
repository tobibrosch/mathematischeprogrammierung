—Anleitung— 
Um das Spiel zu öffnen, sollte es im gleichen Verzeichnis gestartet werden, in dem sich die erforderliche Datei befindet. Andernfalls können möglicherweise keine Bilder geladen werden. Wenn sich die Konsole im richtigen Pfad befindet, kann das Spiel durch Ausführen der Datei "breakout.py" gestartet werden. Nachdem das Programm gestartet wurde, wird der Spieler aufgefordert, einen Namen einzugeben. Sobald ein Name eingegeben wurde, kann das Spiel gestartet werden. Jeder Button wird mit der Maus betätigt. Der Boden wird mit den Tasten A und D bewegt. Mit ESC kann das Programm jederzeit beendet werden. Im Startscreen kann unten rechts auf Settings zugegriffen werden. Dort kann der Schwierigkeitsgrad, das Scoreboard und eine eigene Formation erstellt werden. Mit der Leertaste kann das Spiel pausiert werden.
